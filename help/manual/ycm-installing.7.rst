How to install YCM and use it in Your Project
---------------------------------------------

Download, compile and install the latest version of YCM from:

https://github.com/robotology/ycm/releases

Un pack it on a directory and compilte it:

.. code-block:: bash

    cd ycm-master
    mkdir build
    cd build
    cmake ../
    make
    sudo make install
