Changes made since YCM 0.10.0 include the following.


Modules
=======

Generic Modules
---------------

* :module:`InstallBasicPackageFiles`: The package is now exported to the user
  package registry, when the :cmake:variable:`CMAKE_EXPORT_PACKAGE_REGISTRY` is
  enabled.
